# 🏠 Dragon Real Estate Price Prediction

## 📌 Overview
This project predicts housing prices using machine learning techniques on real estate data. It includes data preprocessing, feature engineering, model building, and evaluation to develop an accurate regression model.

## 🚀 Features
- Data cleaning and preprocessing  
- Handling missing values using median imputation  
- Feature engineering (created new features to improve model performance)  
- Stratified train-test splitting  
- Model training using multiple algorithms  
- Cross-validation for reliable performance evaluation  

## 🧠 Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

## 📊 Evaluation Metrics
- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  
- R² Score  

## 🏆 Results
- Random Forest Regressor performed best among all models  
- Achieved low RMSE indicating good prediction performance  
- Cross-validation helped ensure model stability and reduced overfitting  

## ⚙️ Pipeline
Used Scikit-learn Pipeline for:
- Handling missing values  
- Feature scaling using StandardScaler  

## 🔮 Future Improvements
- Hyperparameter tuning for better accuracy  
- Deployment using Streamlit or Flask  
- Integration with real-time housing data  

## 📚 Key Learnings
- Importance of data preprocessing and feature engineering  
- Use of pipelines for efficient ML workflows  
- Model evaluation using RMSE and cross-validation  
- Understanding regression models in real-world scenarios
